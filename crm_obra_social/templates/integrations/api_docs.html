Documentación de la API

Autenticación

Todas las solicitudes deben incluir el header X-API-Key con la clave generada en esta sección.

X-API-Key: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx

También se acepta como parámetro GET: ?api_key=...

POST {{ base_url }}/api/v1/leads/ — Crear lead

Crea un lead nuevo. Si el teléfono ya existe, actualiza los datos vacíos del lead existente.

Request (JSON)
{
  "nombre_completo": "Juan Pérez",        // requerido
  "telefono": "1123456789",               // requerido
  "codigo_pais": "54",                    // opcional, default: 54
  "email": "juan@example.com",
  "dni": "35123456",
  "localidad": "Buenos Aires",
  "provincia": "Buenos Aires",
  "plan": "Individual",                   // nombre exacto del plan
  "notas": "Interesado por Google Ads",
  "origen": "web",                        // web|campaña|referido|llamada|whatsapp
  "datos_extra": {                        // cualquier campo adicional
    "utm_source": "google",
    "utm_campaign": "marca_2026",
    "formulario": "home_page"
  }
}
Respuestas
201{"status":"created","lead_id":42,"telefono":"+541123456789"}
200{"status":"updated","lead_id":42,"telefono":"+541123456789"}
422{"error":"Teléfono inválido: ...","code":"validation_error"}
401{"error":"API key inválida o ausente","code":"unauthorized"}

POST {{ base_url }}/api/v1/webhook/{source}/ — Webhook genérico

Igual que POST /leads/ pero el campo origen se toma del parámetro {source} de la URL si no viene en el body.

POST {{ base_url }}/api/v1/webhook/landing-google/
# El lead se creará con origen=landing-google (guardado en notas/datos_extra)

Útil para conectar herramientas como Zapier, Make, n8n, formularios de WordPress, Webflow, etc.

Ejemplo con curl
curl -X POST {{ base_url }}/api/v1/leads/ \
  -H "Content-Type: application/json" \
  -H "X-API-Key: TU-API-KEY-AQUI" \
  -d '{
    "nombre_completo": "María García",
    "telefono": "1145678901",
    "email": "maria@gmail.com",
    "origen": "web",
    "datos_extra": {"formulario": "landing_v2"}
  }'
Ejemplo con JavaScript (fetch)
fetch('{{ base_url }}/api/v1/leads/', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'TU-API-KEY-AQUI'
  },
  body: JSON.stringify({
    nombre_completo: document.getElementById('nombre').value,
    telefono: document.getElementById('telefono').value,
    email: document.getElementById('email').value,
    origen: 'web',
    datos_extra: { formulario: 'contacto_home' }
  })
})
.then(r => r.json())
.then(data => console.log('Lead creado:', data.lead_id));
